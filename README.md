
# MEMORIA Host Manager

***Ипользуется для локального селфхоста MEMORIA.***

  

**Сама MEMORIA не заточена только под использование внутри локальной сети, её так же можно захостить любым другим способом.**

## 1.1. Сборка `.exe` файла и добавления домена в hosts

В папке находится скрипт `Installer.bat`. Просто запустите его двойным кликом.

В нём есть два пункта:

- Сбилдить MEMORIA Host Manager (используя Pyinstaller --onefile)

- Настройка hosts

Следуйте инструкциям скрипта.

  

*Примечание:* Если при запуске скомпилированного .exe вы получаете ошибку **"Error loading Python DLL: python310.dll LoadLibrary: Не найден указанный модуль"**, это баг Windows Store версии Python. Лучше всего использовать классический установщик Python скачанный с python.org.

## 1.2. Готовый билд.

Хоть готовый билд уже и есть, для добавления домена всё равно необходимо скачать Installer.bat.

Либо вручную ввести IPv4 хоста в hosts под домен MEMORIA.

  

## 2. Использование Менеджера

Теперь вы можете выбрать корневую папку MEMORIA прямо в приложении с помощью кнопки **"ВЫБРАТЬ ПАПКУ"**. Это позволяет запустить хост из любой директории, где находится Next.js.

  

## 3. Результат

MEMORIA Host Manager будет слушать порт 80 на всех интерфейсах (0.0.0.0:80). Для получения доступа к веб сервису с машины в локальной сети - MEMORIA:80
